Icollect

This code represents a media scraper. Get all Images of a Website of your choice.

The script collects all:
    
    - Img's with attributes: "src", "data-src", "srcset"
    - Div's with attribute: "data-thumbnail"
    - iFrames with attribute: "src"
        - collects all links and passes it to the main function
        
If you click on Download folder, it will immediately download you a zip containing every Picture it collected.
